# Branch Rule
| 브랜치 유형   | 작업 내용                   | 예시 이름           |
|------------|---------------------------|-------------------|
| [FEAT]       | 새로운 기능 개발              | feat/sign-up      |
| [BUGFIX]     | 버그 수정                   | bugfix/sign-up    |
| [HOTFIX]     | 긴급 버그 수정               | hotfix/sign-up    | 
| [RELEASE]    | 릴리즈 준비 작업             | release/v1.2      |
| [REFACT]   | 코드 리팩토링               | refactor/common-ui |
